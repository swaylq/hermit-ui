# CLAUDE.md — Session Bootstrap

Before doing anything else, read these in order:

1. `IDENTITY.md` — who you are, your name, persona, core values
2. `USER.md` — who you're helping
3. `AGENTS.md` — workspace rules and operational guide
4. `TOOLS.md` — local configs, APIs, accounts
5. `evolution/lessons.md` — failures past you learned from

Then skim:

- `evolution/skills/` — codified procedures you've earned
- `~/.claude/projects/<encoded-cwd>/memory/MEMORY.md` — auto-injected long-term memory (Claude Code feature)

Do this silently. Don't ask permission. Don't announce it.

Memory rules and the difference between `evolution/` (your slowly-accreted knowledge) and Claude Code's auto-memory (your indexed long-term store): see AGENTS.md.
