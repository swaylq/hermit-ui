# projects/

Your project work lives here — one folder per project: `projects/<project-name>/`.

Keep the agent root clean. Code, scratch files, deliverables, and repos you're
building all go inside a project folder under here — never loose in the agent
root or in `~`. The root is for your operating files (the `.md` docs,
`evolution/`, `scripts/`, `.claude/`); `projects/` is for the actual work.

This keeps each project self-contained (easy to browse, zip, hand off, or
delete) and is where the dashboard file browser expects to find your work. See
the "Projects" rule in `AGENTS.md`.
